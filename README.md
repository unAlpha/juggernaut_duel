# 剑圣对决 · 概率科普互动演示

一个用于直观理解「闪避」与「暴击」在概率层面优劣的网页小工具。

## 玩法

直接用浏览器打开 `index.html`，无需任何服务器或依赖安装。

调整两名剑圣的参数，点击「开始模拟」观看对决动画，再通过「数据分析」查看概率推演结果。

## 参数说明

| 参数 | 说明 |
|------|------|
| 初始血量 | 角色总血量（1–100） |
| 闪避概率 | 每次被攻击时成功规避的概率 |
| 暴击概率 | 每次攻击命中时触发暴击的概率 |
| 暴击倍数 | 暴击造成的伤害倍率（基础伤害为 1） |

**战斗模式**：同时对砍 / 轮流（闪避/暴击先手）/ 单边攻击

**结算优先级**：闪避计算优先，或暴击触发「克敌机先」（暴击先算，有概率无视闪避）

## 数据分析面板

- **本局复盘**：当前这局的血量变化曲线
- **千局推演**：蒙特卡洛模拟 1000 局，统计双方胜率与存活时平均剩余血量
- **血量-胜率趋势**：在 1–100 血量范围内扫描，展示胜率随血量变化的理论趋势
